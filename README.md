Quick Installation
==================
Use curl (for Mac OS X):
    curl -o - https://raw.github.com/WitzHsiao/my-vimrc/master/install.sh | sh
or wget (for most UNIX platforms):
    wget -O - https://raw.github.com/WitzHsiao/my-vimrc/master/install.sh | sh
